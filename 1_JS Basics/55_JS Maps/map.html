JavaScript Maps

Maps are a new object type in ES6 (2015) that allow you to store key-value pairs. The key can be of any type and the value can be of any type.

Maps are iterable and can iterate in the order of the insertion of the elements.

Maps are similar to objects but there are some differences between them.
